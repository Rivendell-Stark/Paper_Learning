## 论文简要介绍

本文是发表在2024年1月份 *Management Science* 上的文献[*Dog Eat Dog: Balancing Network Effects and Differentiation in a Digital Platform Merger*](https://pubsonline.informs.org/doi/10.1287/mnsc.2023.4675)。这篇文章聚焦数字平台并购中 “网络效应” 与 “平台差异化” 的核心矛盾，将美国两大宠物托管平台（Rover 与 DogVacay）2017 年合并事件作为准自然实验，通过因果识别揭示“网络效应” 与 “平台差异化”的相互作用，为平台战略与反垄断监管提供实证依据。


这篇文章的研究层次是市场-月度层次：考虑到宠物托管市场是一个地理相对隔离的市场，即人们偏好本地交易。因此作者利用邮政编码划分为不同区域的市场，每个区域的市场作为一个样本点进行分析。DID方法要求划分控制组与处理组，作者是按照Rover平台（即合并后保留的平台）在宣布合并前的市场份额为划分依据，大于80%的作为控制组，0~20%、20%~40%、40%~60%、60~80%作为4个处理组进行分析。时间上，作者并没有按照处理前-处理后划分，而是按照月份划分，从宣布合并前一月（2017-2）为基准，一直到合并后6个月（2017-12）。原因在于合并事件不是一次瞬时的冲击，而是一个比较长的时间段。两个平台2017年2月开始达成合并协议，2017年5月宣布关停DogVacay，2017年7月正式关停DogVacay平台。因此作者采用的是一种事件研究的思路，展示了月度的效应变化。


作者使用的实证方法是一种特别的DID方法：与标准的DID通过虚拟变量（Dummy）控制处理组与控制组不同，这篇文章利用协变量倾向得分匹配的方式为处理组中的每个市场找一个类似的在控制组中的市场，以此控制变量。作者使用的协变量是区域内商家数量。匹配之后，将处理组的结果变量减去匹配的控制组的结果变量，以此作为回归方程中左边部分。这样一来，右边其实没有交互项与控制项，而只有表示时间的虚拟变量与截距和误差项。


作者通过这次合并事件证明了网络效应的存在，并说明网络效应不足以支持平台的合并策略。因为即使是两个非常相似的平台，平台合并导致的差异化损失也足以抵消网络效应带来的收益。以至于在市场层面平均来看，用户无论是使用一个还是两个平台，境况都差不多。结合平台价格在合并后并未上涨的事实，研究表明就平均而言，单一平台并未比两个竞争平台提供更大的消费者剩余。除此之外，用户与商家间的重复交易（即用户多次与同一商家交易，而不是经常更换商家）在抵消网络效应方面发挥着重要作用。因此，平台差异化在提供儿童托管的平台上可能更为重要，因为重复交易更为频繁，而在网约车平台上则相对不那么重要，因为重复交易较为罕见。
